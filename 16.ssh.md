# SSH Key

1. Generate SSH Key

    ```bash
    # location public and private RSA key generated.
    # default name will be id_rsa (private), id_rsa.pub (public)

    ls -l ~/.ssh

    # generate ssh key
    ssh-keygen -t rsa
    ```

2. Copy ssh key (public) to remote server

    ```bash
    # ssh-copy-id {user}@{ip/host}
    ssh-copy-id root@192.168.1.1
    ```

    The public key will be copied/added to the server (192.168.1.1) at <code>~/.ssh/authorized_keys</code>

3. Connect to remote server via SSH. You should be auto login to the user(root) home.
    ```bash
    ssh root@192.168.1.1
    ```
    
# SSH Auth to Github
1. Cek apakah sudah ada SSH key
   ```
   ls -al ~/.ssh
   ```

   Kalau muncul file seperti:
   ```
   id_rsa.pub
   id_ed25519.pub
   neutro.github.pub
   ```

   berarti kamu sudah punya SSH key. Kalau belum ada, lanjut ke langkah berikut.
   
2. Generate SSH Key
   ```
   ssh-keygen -t rsa

   ssh-keygen -t rsa                                                                             ─╯
   Generating public/private rsa key pair.
   Enter file in which to save the key (/home/junaidi/.ssh/id_rsa): neutrofoton.github 
   Enter passphrase (empty for no passphrase): 
   Enter same passphrase again: 
   Your identification has been saved in neutrofoton.github
   Your public key has been saved in neutrofoton.github.pub
   The key fingerprint is:
   SHA256: XXXXX
   ```

3. Tambahkan SSH key kamu ke agent:
    ```
    ssh-add ~/.ssh/neutrofoton.github
    ```

4. Tambahkan SSH key ke akun GitHub
   - Copy
      ```
      cat ~/.ssh/neutrofoton.github.pub
      ```
   - Buka https://github.com/settings/keys, Klik “New SSH key”
   - Isi:
      - Title: misal Laptop Dev
      - Key type: Authentication Key
      - Key: paste isi key tadi
      - Klik Add SSH key
5. Uji koneksi SSH ke GitHub
   ```
   ssh -T git@github.com
   
   # Kalau berhasil, muncul pesan seperti:
   Hi neutrofoton! You've successfully authenticated, but GitHub does not provide shell access.
   ```
